# AICC 维护参考

本目录存放 AICC 的维护参考文档。

这些文档不是设计目标。它们是在某个时间点基于 AICC 需求、设计草案和具体实现代码整理出来的，用于帮助理解当前行为、常见操作、实现缺口和历史决策，不能作为后续设计工作的约束来源。

其中 `aicc_acceptance_test_plan.md` 属于维护文档，用于指导当前实现的验收组织和测试覆盖，不作为 AICC 设计目标来源。

修改 AICC 行为时：

- 需求、协议契约、路由设计和验收目标以 `doc/aicc` 根目录文档为准。
- 当前实现以 `src/frame/aicc`、共享 API 类型、UI 调用方和测试为准。
- 只有当这些维护参考仍能帮助后续维护时，才同步更新。
